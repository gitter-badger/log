A learning experiment in mobile application design & development.

Getting Started
---

install:

```sh
$ npm install
```

build assets:

```sh
$ npm run build
```

watch assets:

```sh
$ npm run watch
```

>If you would like to use the watch command, you will need [watchman](https://facebook.github.io/watchman/) installed.

```sh
$ brew update
$ brew install watchman
```

server:

```sh
$ npm start
$ npm stop
```

testing:

```sh
$ npm test
```
