Todoofus
===
A learning experiment in mobile application design & development.

Getting Started
---
This will install most (if not all) of the necessary dependencies.
```sh
$ npm install
```
Building assets:
```sh
$ npm run build-js && npm run build-css
```
Static server:
```sh
$ npm start
```
Stop the server:
```sh
$ npm stop
```
Running the tests:
```sh
$ npm test
```
